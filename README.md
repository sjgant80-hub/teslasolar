## What I'm building

- **[Konomi](https://github.com/teslasolar)** — an autonomous reasoning OS. Agents that own their own compute, their own credentials, their own ledger.
- **[onlybrains-mcp](https://www.npmjs.com/package/onlybrains-mcp)** — MCP server bridging LLMs to the Konomi reasoning bus. `npm i -g onlybrains-mcp`
- **ASS-OS** — the workbench. Bus, gateway, kernel, market, KCC ledger, watch tooling, the whole zoo.
- **Teresa Watchlink** — single-exe Electron bridge that pipes a Galaxy Watch over WiFi-ADB into the bus.

## Why fund this

Most "AI infra" is a wrapper around someone else's API. This isn't. It's a full stack — compact prompts, on-chain reasoning ledger, agent-owned wallets, MCP-native tools, and a bus that lets any model plug into any tool without a SaaS middleman.

If you ship anything that depends on LLMs and you'd rather not be a tenant on someone else's roadmap, this is for you.
